# Calculator

A simple C# console application for performing basic arithmetic operations.

## Features
- Safe input for numbers
- Safe input for operations: `+`, `-`, `*`, `/`, `^`, `%`
- Addition, subtraction, multiplication, division, power, percent
- Division-by-zero handling
- Console-based interface
- Logic separated from UI (`Logic.cs` handles calculations, `UI.cs` handles input/output)
- Entry point is Program.cs

## Requirements
- .NET 6.0 or newer
- C# 10 or newer
